让盒子居中的四种方法

让盒子居中的方法

第一种方法position+margin

#box{

        position: fixed;(也可以是absolute)
        left : 50%;
        margin-left: -盒子宽度的一半;
        top:50%;
        margin-top: -盒子高度的一半;
    }

第二种方法position+margin的第二种方式

#box{

       position: fixed;(也可以是absolute)
       top:0;
       left: 0;
       right: 0;
       bottom:0;
       margin: auto;
   }

第三种方法需要借助伪元素来撑开高度

html,body{
       height:100%;
   }
   body{
       text-align: center;
   }
   body::before{
       /* 必备属性 */
       content : '';
       display: inline-block;
       height:100%;
       vertical-align: middle;
   }
   #box{
       vertical-align: middle;
       display: inline-block;
   }

第四种方法伸缩盒子布局

html,body{
       height: 100%;
   }
   body{
       /* 开启伸缩盒子 */
       display: flex;

       /* (主轴)水平方向对齐 */
       justify-content: center;
       /* （侧轴）垂直方向对齐 */
       align-items: center;
   }
